Mi primer CRUD

Bienvenido! En este proyecto usted podrá realizar un registro de tareas asignándole una descripción, así también podrá editar o eliminar dicha tarea si así lo requiere. Todo esto basado en el acrónimo CRUD (Create Read Update Delete) el cual nos permite realizar funciones básicas para el manejo de información y su persistencia.
